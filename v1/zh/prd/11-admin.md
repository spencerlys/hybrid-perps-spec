# L9 后台管理层

## 路由与对赌配置

| 功能 | 说明 |
|------|------|
| 全局路由阈值 | 默认 $10,000，实时生效 |
| 按币种路由阈值 | 每个币种独立设置 |
| 对赌白名单 | 初期建议仅 BTC/ETH |
| 极端行情开关 | 一键暂停对赌（全部路由到 HL） |
| 客损准备金比例 | 默认 20%，可调整 |

## 风控仪表盘

### 实时指标

| 面板 | 内容 |
|------|------|
| 对赌净敞口 | 各币种实时敞口 + 对冲状态 |
| 平台 B-book PnL | 日/周/月收益曲线 |
| 路由统计 | INTERNAL vs HYPERLIQUID 笔数/金额/占比 |
| 清算统计 | 今日清算数量、金额、分 INTERNAL/HL |
| HL 账户状态 | 保证金率、可用余额、各仓位规模 |
| 风险准备金 | 余额、近期变化、预警状态 |

### 计算偏差面板

| 指标 | 说明 |
|------|------|
| 当日偏差总额 | 日累计 平台 计算 vs HL 实际的差额 |
| 偏差趋势图 | 近 30 天每日偏差折线 |
| 按币种偏差排行 | 哪些币种偏差最大 |
| 最近告警列表 | 偏差率 > 1% 的交易记录 |

### 归集与流动性面板

| 指标 | 说明 |
|------|------|
| 各链热钱包余额 | TRON / ETH / Solana 余额 |
| 归集历史 | 近期归集交易记录、Gas 消耗 |
| 提现队列 | 待处理提现数量 + 金额 |
| 流动性覆盖率 | 热钱包 / 用户总负债比 |

## 用户与订单查询

| 功能 | 说明 |
|------|------|
| 用户仓位查询 | 含 INTERNAL / HYPERLIQUID 标记 |
| 订单路由历史 | 每笔订单的路由决策记录 |
| 清算记录 | 清算详情、保证金归零记录 |
| 计算偏差记录 | HL 回执 vs 平台 计算的差异明细 |
| 手动余额调整 | 需审批工作流，有完整审计日志 |

## 权限管理（RBAC）

| 角色 | 权限范围 |
|------|---------|
| Super Admin | 全部权限 |
| Risk Manager | 对赌开关、敞口配置、对冲参数、准备金、偏差阈值调整 |
| Operations | 用户查询（只读）、提现审批、归集手动触发 |
| Data Analyst | 报表（只读）、数据导出 |

## 审计日志

所有后台操作均需记录：
- 操作人、时间、IP
- 操作类型和变更内容
- 审批流程记录（如余额调整、冷钱包出金审批）
- 日志不可删除/修改（append-only）
