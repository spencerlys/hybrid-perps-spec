# L9 后台管理层

## 路由模式控制页面

### 模式开关

管理员可在此页面切换三种路由模式（需 Risk Manager 或以上权限）：

```
当前模式：[ HL模式 ]  [ 正常模式 ✓ ]  [ 对赌模式 ]

⚠️ [仅 HL 模式下显示的橙色横幅]
当前净敞口过高，请尽快到 Hyperliquid 进行对冲！
```

### 数值配置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 正常模式路由阈值 | $10,000 | NORMAL_MODE 下，≤ 此值走对赌 |
| 对赌模式路由阈值 | $50,000 | BETTING_MODE 下，≤ 此值走对赌 |
| HL 模式触发净敞口 | $800,000 | 净敞口 ≥ 此值时触发切换建议/自动切换 |
| 对赌模式触发净敞口 | $50,000 | 净敞口 ≤ 此值时触发切换建议/自动切换 |
| 当前净敞口（只读） | 实时刷新 | 各币种合计净敞口绝对值，5 秒刷新 |
| 自动模式切换 | 关闭 | 开启后净敞口触达阈值时自动切换并通知 |
| 按币种路由阈值 | 继承全局 | 每个币种可单独覆盖全局阈值 |

页面底部显示近 30 次切换记录：

## 风控仪表盘

### 实时指标

| 面板 | 内容 |
|------|------|
| 对赌净敞口 | 各币种实时敞口 + 对冲状态 |
| 平台 B-book PnL | 日/周/月收益曲线 |
| 路由统计 | INTERNAL vs HYPERLIQUID 笔数/金额/占比 |
| 清算统计 | 今日清算数量、金额、分 INTERNAL/HL |
| HL 账户状态 | 保证金率、可用余额、各仓位规模 |
| 风险准备金 | 余额、近期变化、预警状态 |

### 计算偏差面板

| 指标 | 说明 |
|------|------|
| 当日偏差总额 | 日累计 平台 计算 vs HL 实际的差额 |
| 偏差趋势图 | 近 30 天每日偏差折线 |
| 按币种偏差排行 | 哪些币种偏差最大 |
| 最近告警列表 | 偏差率 > 1% 的交易记录 |

### 归集与流动性面板

| 指标 | 说明 |
|------|------|
| 各链热钱包余额 | TRON / ETH / Solana 余额 |
| 归集历史 | 近期归集交易记录、Gas 消耗 |
| 提现队列 | 待处理提现数量 + 金额 |
| 流动性覆盖率 | 热钱包 / 用户总负债比 |

## 用户与订单查询

| 功能 | 说明 |
|------|------|
| 用户仓位查询 | 含 INTERNAL / HYPERLIQUID 标记 |
| 订单路由历史 | 每笔订单的路由决策记录 |
| 清算记录 | 清算详情、保证金归零记录 |
| 计算偏差记录 | HL 回执 vs 平台 计算的差异明细 |
| 手动余额调整 | 需审批工作流，有完整审计日志 |

## 权限管理（RBAC）

| 角色 | 权限范围 |
|------|---------|
| Super Admin | 全部权限 |
| Risk Manager | 对赌开关、敞口配置、对冲参数、准备金、偏差阈值调整 |
| Operations | 用户查询（只读）、提现审批、归集手动触发 |
| Data Analyst | 报表（只读）、数据导出 |

## 审计日志

所有后台操作均需记录：
- 操作人、时间、IP
- 操作类型和变更内容
- 审批流程记录（如余额调整、冷钱包出金审批）
- 日志不可删除/修改（append-only）
